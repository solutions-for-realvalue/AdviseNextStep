## ğŸ“¦ Installation

To get a local copy up and running, follow these steps:

1. **Clone the Repository**
   
   ```bash
   git clone https://github.com/solutions-for-realvalue/AdviseNextStep.git
   ```

2. **Navigate to the Folder**

   ```bash
   cd AdviseNextStep
   ```

3. **Install NPM Packages**

   ```bash
   npm install
   ```

4. **Start the Server**

   ```bash
   npm start
   ```
